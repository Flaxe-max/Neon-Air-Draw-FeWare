# 🌌 Neon Air Draw — AI Spatial Interface

## English

A high-performance, real-time AI air drawing web application that utilizes advanced hand tracking for a "Minority Report" style spatial interface. 

Draw in the air with your dominant hand and manipulate your creations in real-time with your non-dominant hand using intuitive gestures (Move, Scale, Rotate).

### ✨ Key Features
*   **✋ Dual-Hand Interaction**: 
    *   **Right Hand (Dominant)**: Handles drawing, erasing, and clearing.
    *   **Left Hand (Secondary)**: Handles Move, Scale, and Rotate transformations.
*   **🌍 Multi-Language Support**: Full support for English and Turkish (TR).
*   **🕶️ Premium UI**: Sleek glassmorphism interface with real-time feedback.
*   **⚡ High Performance**: Optimized for fluid 60FPS interactions using MediaPipe.
*   **📖 Gesture Guide**: Built-in interactive manual (How it works?).

### 🛠️ Tech Stack
*   **Frontend**: React + Vite
*   **Hand Tracking**: @mediapipe/hands
*   **Animations**: Framer Motion
*   **Icons**: Lucide React
*   **Styling**: Tailwind CSS + Custom Vanilla CSS

### 🎮 Gesture Manual
| Hand | Gesture | Action |
|---|---|---|
| **Right** | ☝️ Index Up | Start drawing |
| **Right** | 🤏 Pinch | Eraser |
| **Right** | ✊ Fist | Clear all |
| **Left** | ✌️ Two Fingers | Move stroke |
| **Left** | 🤏 Pinch | Scale stroke |
| **Left** | 🖐️ Open Palm | Rotate stroke |

### 🚀 Getting Started
1. `npm install`
2. `npm run dev`

## Türkçe

Gelişmiş el takibi algoritmalarını kullanan, "Minority Report" tarzı bir uzamsal arayüz sunan, yüksek performanslı ve gerçek zamanlı bir yapay zeka hava çizim web uygulamasıdır.

Baskın elinizle havada çizim yapın ve diğer elinizle sezgisel hareketler (Taşıma, Ölçeklendirme, Döndürme) kullanarak çizimlerinizi gerçek zamanlı olarak yönetin.

### ✨ Temel Özellikler
*   **✋ Çift El Etkileşimi**:
    *   **Sağ El (Baskın)**: Çizim yapma, silme ve temizleme işlemlerini yönetir.
    *   **Sol El (Yönetim)**: Taşıma, Ölçeklendirme ve Döndürme işlemlerini yönetir.
*   **🌍 Çoklu Dil Desteği**: Tam Türkçe ve İngilizce desteği.
*   **🕶️ Premium Arayüz**: Gerçek zamanlı geri bildirim sunan şık glassmorphism tasarımı.
*   **⚡ Yüksek Performans**: MediaPipe kullanılarak 60FPS akıcı etkileşimler için optimize edildi.
*   **📖 Yardım Paneli**: Entegre "Nasıl Çalışır?" rehberi.

### 🛠️ Kullanılan Teknolojiler
*   **Frontend**: React + Vite
*   **El Takibi**: @mediapipe/hands
*   **Animasyonlar**: Framer Motion
*   **İkonlar**: Lucide React
*   **Stil**: Tailwind CSS + Özel CSS

### 🎮 Hareket Rehberi
| El | Hareket | İşlem |
|---|---|---|
| **Sağ** | ☝️ İşaret Parmağı | Çizime başla |
| **Sağ** | 🤏 Kıstırma | Silgi |
| **Sağ** | ✊ Yumruk | Hepsini temizle |
| **Sol** | ✌️ İki Parmak | Çizgiyi taşı |
| **Sol** | 🤏 Kıstırma | Ölçeklendir |
| **Sol** | 🖐️ Açık Avuç | Döndür |

### 🚀 Kurulum
1. `npm install`
2. `npm run dev`
